Оператор **BREAK** - создание [действия](Действия.md), реализующего [прерывание цикла](Прерывание_BREAK.md).

### Синтаксис

    BREAK

### Описание

Оператор **BREAK** создает действие, которое выходит из наиболее вложенного цикла, внутри которого находится.
